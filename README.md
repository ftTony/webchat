#### 2.1 环境准备

要运行**智能聊天系统**，你需要`Node.js(V10 LTS版本，例如10.6.3)`开发环境和`MySql`数据库

- 安装 Node.js
  - 官网[http://nodejs.cn/download/](http://nodejs.cn/download/)
  - 更推荐使用 nvm 安装 Node.js
    - 安装 nvm
    - 通过 nvm 安装 Node.js
- 安装 MySql

推荐在 Linux 和 MacOS 系统中运行，Windows 系统也兼容的，但是遇到意外问题会多一些
